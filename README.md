# Mizzou Final Exam Scheduler

A streamlined web application that helps University of Missouri students easily view their final exam schedule, eliminating the need to navigate the complex official registrar website.

**Live Demo:** [mizzou-final-exam-scheduler.vercel.app](https://mizzou-final-exam-scheduler.vercel.app/)

## âœ¨ Features

- **Simple Interface**: Input your course details and get your final exam schedule instantly
- **Time-Saving**: Avoids the multi-step navigation typically required on the registrar's website
- **Clear Display**: Presents exam dates, times, and locations in a clean, easy-to-read format
- **Full-Stack Application**: Built with a React frontend and a Node.js/Express backend
- **Free & Accessible**: No login or student account required to use

## ðŸš€ Getting Started

### Prerequisites
- Node.js (v14 or later)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   git clone https://github.com/EnobongOffiong/Mizzou-Final-Exam-Scheduler.git
   cd Mizzou-Final-Exam-Scheduler

2. **Install backend dependencies**
   cd backend
   npm install

3. **Install frontend dependencies**
   cd ../frontend/mizzou-scheduler
   npm install

### Running Locally

1. **Start the backend server (from the backend directory)**
   npm start

2. **Start the frontend development server (from the frontend/mizzou-scheduler directory)**
   npm start
3. Open your browser and navigate to http://localhost:3000 to use the application
